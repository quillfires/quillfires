### About

I write Python - async-first, full-stack when needed. I build APIs, payment integrations,
and production web applications. Professionally I develop and maintain custom software
for businesses of all sizes: HR modules, POS systems, vessel tracking solutions, and
news portals like [hurihaa.mv](https://hurihaa.mv), one of the Maldives' leading news
sites, through my brand [Lagoon Labs](https://lagoonlabs.co).

I also work with TypeScript and PHP when the project calls for it. Based in 🇲🇻 Maldives.

### Open Source

I build tools that solve real problems I encounter in production.
Most of my open source work focuses on backend utilities,
developer tooling, and payment integrations.

<table>
<tr>
<td width="50%" valign="top">

**[bml-connect-python](https://github.com/quillfires/bml-connect-python)**

Python SDK for the Bank of Maldives Connect API v2 providing a complete
developer integration layer for Maldivian payment processing.

Supports all four payment integration methods: Redirect, Direct,
Card-On-File Tokenization, and PCI Merchant Tokenization.

Includes both synchronous and asynchronous clients.

**[SecurePasswordGenerator](https://github.com/quillfires/SecurePasswordGenerator)**

Python CLI tool and library for generating secure, high-entropy
credentials.

Supports flexible password generation with custom character sets,
cryptographic keys, and security-focused credential formats for
real-world applications.

Includes generators for strong passwords, WEP/WPA keys, CodeIgniter
encryption keys, and other high-entropy secrets used in infrastructure
and application security.
